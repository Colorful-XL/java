12#局部变量 ：方法或语句块内部定义的变量
	使用前必须初始化
#实例变量(成员变量)//属于类，与方法无关
方法外部，类内部定义的变量
如果不进行初始化他会自动初始化为该类型的默认初始值（布尔默认为false）
#常量 final 只能被赋值一次
final int c= ;//再赋值出错
#二元运算符
1.整数运算
如果两个操作数中有一个为long，则结果也为long
没有long时，结果为int。即使操作数全为short，byte，结果也是int
2.浮点运算
如果两个操作数中有一个为double，则结果也为double
只有两个操作数都是float时结果才为float
#逻辑与逻辑或操作的是bool变量
#按位与按位或按位异或是对位操作
~取反  &按位与 |按位或 ^按位异或（不一样为1一样为0）
<< 左移 ：每左移一位相当于X2 //	3<<2 == 3*2*2;
>> 右移 ：每左移一位相当于/2
# +两边只要有一边为字符(被双引号括住)则+变为strcat字符串连接函数//4+"5"=45;
